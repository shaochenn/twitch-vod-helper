## English
A lightweight Chrome extension to customize seek/skip intervals on Twitch VODs.

### 🚀 Features
- **Custom Seek Time:** Change the default 10s skip to **5s** for more precise control.
- **Native Experience:** Works perfectly with Arrow Left/Right keys. `→`

一個輕量級的 Chrome 擴充功能，讓你能自定義 Twitch VOD 的跳轉秒數。
