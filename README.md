<a name="english"></a>
## 🌍 English

A lightweight Chrome extension to customize seek/skip intervals on Twitch VODs.

### 🚀 Features
- **Custom Seek Time:** Change the default 10s skip to **5s** (or your preference) for more precise control.
- **Native Experience:** Works perfectly with <kbd>←</kbd> <kbd>→</kbd> Arrow keys.
- **Support:** If this helps you, feel free to [Buy me a coffee ☕️](https://ko-fi.com/davyz)

##  中文說明

一個輕量級的 Chrome 擴充功能，讓你能自定義 Twitch VOD 的跳轉秒數。

### 🚀 核心功能
- **自定義跳轉：** 將原本固定的 10 秒跳轉改為更精準的 **5 秒**（或任何你想要的秒數）。
- **原生操作：** 完美相容鍵盤 <kbd>←</kbd> <kbd>→</kbd> 左右鍵，不影響原有操作習慣。
- **支持開發：** 如果這個工具有幫到你，歡迎 [請我喝杯咖啡 ☕️](https://ko-fi.com/davyz)
